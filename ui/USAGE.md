## Usage

### Quick Start

This is all you need to get started.

```html
<!DOCTYPE html>
<cxl-application>
	<cxl-appbar>
		<cxl-navbar><cxl-item>Home</cxl-item></cxl-navbar>
		<cxl-appbar-title>Application Title</cxl-appbar-title>
	</cxl-appbar>
	<cxl-content>
		<cxl-t h1>Hello World</cxl-t>
	</cxl-content>
</cxl-application>
```
